# Agentforce 360 Challenge Repository

This repository contains the **Apex classes**, **Lightning Web Components (LWC)**, configuration notes, and supporting code used to complete the six challenges in the **Agentforce 360 Challenge**.

The purpose of this repository is to organize all challenge-specific Salesforce development work in one place, making it easier to review, deploy, test, and extend the solutions built across Apex, LWC, Agentforce, Slack, Voice, and Data 360 use cases.

---

## Repository Overview

The repository includes code and implementation references for the following six challenges:

1. **Challenge 1 - Apex and LWC**
2. **Challenge 2 - Agentforce Vibes**
3. **Challenge 3 - Holiday Challenge**
4. **Challenge 4 - Slack**
5. **Challenge 5 - New Agentforce Builder and Agentforce Voice**
6. **Challenge 6 - Data 360**

Each challenge focuses on a different business scenario and explores different capabilities of the Salesforce platform and Agentforce ecosystem.

---

## Challenge Summary

### Challenge 1 - Apex and LWC

**Theme:** Retail and Consumer Goods

This challenge focuses on the latest capabilities in **Lightning Web Components** and **Apex**.

Key concepts covered:

- State management in Lightning Web Components
- Apex formulas
- Apex Cursors
- Targeted Apex test execution
- Apex and LWC integration
- Building an agent for a retail and consumer goods business

The implementation includes Apex classes and LWC components that demonstrate how Salesforce UI and backend logic can work together to support intelligent business workflows.

---

### Challenge 2 - Agentforce Vibes

**Theme:** Healthcare and Life Sciences

This challenge focuses on improving development productivity using **Agentforce Vibes**.

Key concepts covered:

- Faster coding with Agentforce Vibes
- Testing and debugging support
- Model Context Protocol tools
- Agentforce rules configuration
- Central prompt library usage
- Building an agent for a healthcare and life sciences business

The implementation demonstrates how Agentforce Vibes can assist developers in writing, testing, and debugging Salesforce solutions more efficiently.

---

### Challenge 3 - Holiday Challenge

**Theme:** Holiday Season Agent

This bonus challenge guides the development of a holiday-themed agent.

Key concepts covered:

- Seasonal agent design
- Customer interaction use cases
- Holiday-specific prompts and actions
- Personalized recommendations
- Agent response customization

This challenge demonstrates how Agentforce can be adapted for seasonal campaigns, holiday support scenarios, and festive customer engagement.

---

### Challenge 4 - Slack

**Theme:** Automotive Business

This challenge focuses on connecting **Salesforce with Slack** and bringing Agentforce into team collaboration workflows.

Key concepts covered:

- Salesforce and Slack integration
- Using agents inside Slack workflows
- Agent-assisted collaboration
- Automotive business use cases
- Real-time interaction between teams and Salesforce data

The implementation shows how agents can support business users directly inside Slack, helping teams act faster without switching between multiple systems.

---

### Challenge 5 - New Agentforce Builder and Agentforce Voice

**Theme:** Financial Services

This challenge focuses on the new **Agentforce Builder** experience and **Agentforce Voice**.

Key concepts covered:

- Designing agents using the new Agentforce Builder
- Creating and managing agent scripts
- Implementing Agentforce Voice
- Voice-based agent interactions
- Financial services business use cases

The implementation demonstrates how voice-enabled agents can improve customer and employee experiences in financial services scenarios.

---

### Challenge 6 - Data 360

**Theme:** Manufacturing Business

This challenge focuses on working with structured and unstructured data using **Data 360**.

Key concepts covered:

- Data ingestion
- Structured data processing
- Unstructured data processing
- Using processed data in Apex
- Using processed data in Flows
- Building an agent for a manufacturing business

The implementation demonstrates how processed enterprise data can be used across Salesforce automation, Apex logic, and Agentforce-powered experiences.
